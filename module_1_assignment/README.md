# Assignements for Module #1: C++ from Robotics Perspective

#### How to Run
### Building and Running

- **Go:**

```sh
cd module1_assignement
```
- **Create a directory**
```sh
mkdir build
```
- **Change of directory**
```sh
cd build
```
- **Execute cmake**
```sh
cmake ..
```
- **Execute make**
  ```sh
  make
  ```

- **Run the task_1:**
  ```sh
  ./robot_r
  ```

- **Run the task_2:**
  ```sh
  ./robot_sensor
  ```

- **Run the task_3:**
  ```sh
  ./sensor_r
  ```
